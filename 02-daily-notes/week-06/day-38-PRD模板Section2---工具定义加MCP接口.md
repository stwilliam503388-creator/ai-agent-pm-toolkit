## 核心概念

## 1. 工具定义模板

```
## 工具定义
| 工具名 | 功能 | 输入 | 输出 | 权限级别 | 调用频率限制 |
|--------|------|------|------|---------|------------|
| search | 搜索 | query | results | 只读 | 100次/分钟 |
| create | 创建 | data | id | 写入 | 10次/分钟 |
| delete | 删除 | id | status | 执行(需确认) | 5次/分钟 |
```

## 2. MCP接口设计

如果是MCP Server：
- 每个工具对应一个MCP Tool定义
- 输入参数使用JSON Schema描述
- 输出定义包含成功和失败的返回格式
- 权限分级：只读/写入/执行
