## 核心概念

## 1. 评估指标模板

```
## 评估指标体系
| 指标 | 定义 | 目标值 | 测量方式 | 频次 |
|------|------|--------|---------|------|
| 任务完成率 | 用户目标达成的比例 | >85% | 用户反馈+自动检测 | 每日 |
| 幻觉率 | 输出包含事实错误的占比 | <5% | 人工抽样+自动检测 | 每周 |
| 用户满意度 | NPS或满意度评分 | >4.0/5 | 交互后弹窗 | 实时 |
| 单次成本 | 每次交互的Token成本 | <$0.05 | API计费日志 | 每日 |
```

## 2. A/B测试设计

```
## A/B测试计划
- 对照组A：当前方案
- 实验组B：新方案
- 核心假设：B方案在指标X上显著优于A
- 样本量：每组至少N个用户
- 持续时间：至少M天
- 判定标准：p < 0.05 且 效应量>5%
```
