# Day 32 — P4动手Day1 - Multi-Agent写作系统

## 今日目标

用CrewAI定义4个Agent（研究员/写手/审稿人/协调者），搭建Multi-Agent写作系统骨架。

---

## 核心概念

## 1. 项目目标

搭一个4-Agent协作写作流程：
- Agent 1 研究员：搜索信息，输出摘要
- Agent 2 写手：根据摘要写初稿
- Agent 3 审稿人：检查事实错误和语气
- Agent 4 协调者：汇总最终输出

## 2. 技术栈

CrewAI (pip install crewai)

## 3. 今日任务

- 定义4个Agent的Role/Goal/Backstory
- 定义4个Task及执行顺序
- 写 agents.py（Agent定义）和 tasks.py（Task定义）
